LegendRP

NOTICIAS

Este servidor esta hecho con fines educativos, es un proyecto creado por dos alumnos del Instituto Manuel Vazquez Montalban de los ciclos formativos 2nASIR y 2nDAW.



Pronto tendremos nuevas actualizaciones para nuestro segundo proyecto, dispondremos de una pagina web completamente operativa con acceso a nuestro discord de soporte y un bot para poder monitorizar el servidor para discord.



Es un proyecto que nos hemos inspirado en el ya que nos pasamos mucho tiempo jugando diferentes servidores y nos motiva la idea de poder crear nuestro servidor hecho por nosotros. Siempre pueden haber mejoras para el servidor y tiene muchas variables de diferentes scripts para poder ser completamente diferente a otros servidores.
